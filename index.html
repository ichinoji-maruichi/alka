Requirements

動作環境・注意事項

🌐
Google Chrome 114 以上

Chrome専用の拡張機能です。他のブラウザには対応していません。

🔑
Gemini API キー（必須）

Google AI Studio で無料取得できます。初回起動時に設定が必要です。

⚠️
本ツールはGoogle Gemini APIを使用します。APIキーの取得・登録が完了するまでご利用いただけません。

Download

ダウンロード

最新バージョン v1.0.0 をダウンロードしてご利用ください。

📦
拡張機能版（Chrome Extension）
拡張機能版をダウンロード (.zip)

Chromeのサイドバーに常駐またはウィンドウ表示で使う標準版です。Webページを見ながら同時に使えます。

🖥️
Web版（Standalone）
Web版をダウンロード (.zip)

ZIPを解凍してindex.htmlをブラウザで開くだけで使えます。Chrome拡張のインストール不要ですが、サイドバーへの常駐機能はありません。Chrome以外のブラウザでの動作は未確認のため非推奨です。

拡張機能版のインストール手順

1
ZIPを解凍する
ダウンロードした ZIP ファイルを任意のフォルダに解凍してください。
2
Chrome の拡張機能ページを開く
Chromeのアドレスバーに chrome://extensions/ と入力して Enter を押してください。
3
デベロッパーモードをオンにする
右上の「デベロッパーモード」トグルをオンにし、「パッケージ化されていない拡張機能を読み込む」をクリックしてください。
4
解凍したフォルダを選択する
解凍したフォルダ（manifest.json が入っているフォルダ）を選択します。ツールバーにAlkaのアイコンが表示されれば完了です。
📦
Web版のインストール
Web版はZIPを解凍して、フォルダ内の index.html をChromeで開くだけで使えます。

Features

主な機能

✨
プロンプト生成

日本語を入力するとAIが英語の画像生成プロンプトタグに変換します。

✏️
プロンプト改修

既存の英語プロンプトを日本語で指示して修正・改善します。

🎨
創作モード

キーワードからAIがイラスト向けの日本語シーン描写を生成します。

📝
添削モード

日本語のシーン描写をAIイラスト生成に最適化した表現に整形します。

🖼️
画像解析

画像を読み込ませると、再現用の英語プロンプトを逆生成します。

📚
単語辞書

よく使うタグを辞書登録。カテゴリ・フラグで管理・検索できます。

🧪
タグ調合

辞書から複数タグをパレットに追加し、強度・順序を調整して組み立てます。

🎭
マスコット設定

プロファイルでキャラを設定。創作・コメントに個性が反映されます。

📖
詳細な使い方はアプリ内の「マニュアル / このソフトについて」をご参照ください。
拡張機能版：インストール後、ポップアップメニューの「マニュアル / このソフトについて」から開けます。

Setup

Gemini API キーの取得と設定

1
Gemini APIキーを取得する
Google AI Studio にアクセスし、Googleアカウントでログイン後、「APIキーを作成」からキーを発行してください。無料枠でも利用できます。
2
設定画面を開く
ブラウザのツールバーにある拡張機能アイコンをクリック → ポップアップメニュー下部の「設定」ボタンを押します。
3
APIキーを入力して保存
設定画面の「基本設定」タブでAPIキー欄に取得したキーを貼り付け、「保存」ボタンを押します。
4
使用モデルを選ぶ（任意）
設定画面でモデルを切り替えられます。速度重視なら gemini-flash-lite-latest、品質重視なら gemini-pro-latest などが選択肢です。
💡
APIキーの無料枠について：Google AI Studio の無料枠では1分あたりのリクエスト数に制限があります。短時間に何度も生成するとエラーになることがあります。
